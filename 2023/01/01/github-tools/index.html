推荐几个工具，Github 使用体验直线上升！

发表于2023-01-01|更新于2023-02-19
|阅读量:|评论数:

嗨，大家好！这里是BROKENTheBoy，在这里先祝大家新年快乐！在使用 Hexo+Github 制作博客的教程中，有些人可能会因为国内对 Github 的限流而导致在进行hexo d的操作时无法部署的错误 （虽然能打开这个网站的大多数人都能正常部署）。那么今天这篇文章就向大家推荐几个软件，各能解决 Github 被限流问题与日常写代码的问题！还等什么呢？让我们开始吧！

今日推荐软件

  1. 解决限流问题：
    • FastGithub
    • Watt Toolkit
  2. 日常写代码问题：
    • Visual Studio Code

开始推荐

解决限流问题

  1. FastGithub
    FastGithub 是我现在常用的解决限流问题的软件。新版本的 FastGithub 拥有了 UI 界面，操作也十分简单，万分推荐！
    同时，你还可以将 FastGithub 的主文件添加到服务中。
  2. Watt Toolkit (原 Steam++)
    Watt Toolkit 即原来的 Steam++，虽说是用来辅助游戏的，但软件本身也自带对 Github 的加速功能！良心软件实锤！

日常写代码问题

  1. 平常写代码我就只有一个推荐：Visual Studio Code。这个软件为微软官方制作，软件本身可以支持多种编程语言，你可以通过添加扩展来完整体验这款编程软件的强大作用！没有插件的 VSCode = 裸奔
    Visual Studio Code 对于 Git 及 Github 的支持也很好，Git 功能也是内置的，可以一键部署到 Github 仓库，但是 Hexo 使用一键部署插件的还是老老实实 hexo clean && hexo g && hexo d 吧。个人觉得直接使用 Git 内置命令行好用

    感谢大家查看这篇文章！
推荐几个工具，Github 使用体验直线上升！
